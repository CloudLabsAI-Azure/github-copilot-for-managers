# GitHub Copilot for Managers - V2 Lab Files

## Overview

This folder contains the **updated workshop content** based on customer feedback received December 2025. The key changes address the following feedback:

> "The session was more focused on Power BI than GitHub Copilot, which left some participants wanting more practical guidance on Copilot features, governance, and best practices."

## What's Changed

| Change | Description |
|--------|-------------|
| **New Governance Exercise** | Exercise 1 now covers Copilot features, policies, content exclusions, and best practices |
| **Rebalanced Content** | Less Power BI formula creation, more interpretation and action planning |
| **Extended Duration** | Workshop extended from 1 hour to 4 hours |
| **Manager Insights** | Every section includes callouts explaining what metrics mean for decisions |
| **Executive Presentation** | Added frameworks for presenting to leadership |
| **Follow-up Resources** | New resources.md with learning paths and next steps |
